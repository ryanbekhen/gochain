# GoChain

GoChain is a library for LLM (Large Language Model) for Go. It is designed to be simple and easy to use.

## Features

- [x] Function calls

## LLM Support

- [x] Ollama
- [x] Cloudflare Workers AI
- [ ] GPT

## Installation

```bash
go get github.com/ryanbekhen/gochain
```
